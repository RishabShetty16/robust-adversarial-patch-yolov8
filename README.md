# Robust Physical Adversarial Patch Attack for YOLOv8

## Overview

This repository implements a robust physical adversarial patch attack against YOLOv8 using Expectation over Transformation (EOT).

The project consists of four stages:

- Stage A - Digital Patch Optimization
- Stage B - Physical Transfer
- Stage C - Defense Module
- Stage D - Evaluation
